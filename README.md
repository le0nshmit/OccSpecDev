# Occupational Specialism | Greenfield Local Hub   

My Occupational Specialism exam project for a agricultural co-operative called 'Greenfield Local Hub'

## Goals

- Allow producers to list produce for sale
- Allow producers to update stock quantity/information
- Allow producers to view ongoing/past orders

- Allow consumers to browse products with search & filtering
- Allow consumers to favourite, compare and order products
- Allow consumers to create an account
